Printers

A table consisting of the 5 most popular printers. The first 3 being filament and the last 2 being resin.

Filament Printers

  • Creality Ender 3 is a perfect printer for beginners. Not only is it cheap, but it also can perform high quality prints if it is set up properly. It prints 1 spool of filament at a time and prints at an average speed. It is open source so any slicer can be used with it. Link
  • Bambu x1 Carbon is a filament printer for advanced users. The carbon offers the ability to print from 4 different spools at the same time. So you can have multi colored prints without painting them. However it is very expensive and has a huge learning curve to it. Link
  • The Ankermake M5 printer is a great printer for someone who wants to spend the money. It is very high quality and is in the middle ground cost wise. It prints 1 spool at a time, however it is really fast and delivers amazing quality prints. It is very popular in the industry. Link

    • Resin Printers

  • The Elegoo Mars 3 is a perfect beginner resin printer. Although resin printing is not easy, this printer makes it as easy as it can get. You can make extremely high quality minifigures due to its 4K screen. It isn't to expensive, however there can potentially be added costs like a curing station and a wash station as well.Link
  • The Elegoo Saturn 2 is also an amazing resin printer. It can be seen as the bigger version of the mars 3. On the chart below you can see the difference in the two build plate sizes. If you want to make bigger prints this would be perfect for you. Link
